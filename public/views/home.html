The goal is to manage SCRUM projects.


  • You can list all of the projects on the "Project" menu. Then you can click on one project to open it and view details
  • The product backlog is then displayed, you can add user story and edit them
  • In the "Sprint" menu you can view all the sprints of the project with the start date and dead line
  • You can add user story to each sprint.
  • If you click on one sprint, you can add tasks to one sprint on a particular user story
  • You can view the sprint Kanban, by default all tasks created are in the TODO list
  • You can drag'n'drop tasks from TODO to ONGOING or DONE
  • The "My team" menu is the place where you can add member to your project
  • In the "Traceability" menu you can add the ID of commit of each US
